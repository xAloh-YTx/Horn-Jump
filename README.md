# Payback²: The Battle Sandbox | Horn Jump Script
GameGuardian Lua Script made for Payback². Creator: Aloh<br><br>
You can watch the demo [here](https://youtu.be/DWIT8V1_BW8).

## Details
- Whenever you hit the (Horn) button. Your car will Jump.

### Tested on:
- Pb2 v2.106.16 (32bit)
- Pb2 v2.106.16 (64bit)

Didn't Had chance to explore other versions, although this is the latest version.`[01-12-25]`
